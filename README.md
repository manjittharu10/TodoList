# TodoList
A simple To-Do List application built with JavaScript, HTML, and CSS. Add, delete, and mark tasks as complete.
## Features
- Add new tasks
- Delete tasks
- Mark tasks as complete
- Persistent layout for easy task management

## How to Use
1. Open `index.html` in your browser.
2. Type a task in the input field and click **Add**.
3. Click the **checkbox** to mark tasks as complete.
4. Click the **delete icon** to remove a task.

- JavaScript

- CSS
